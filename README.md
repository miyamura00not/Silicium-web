## Overview

Silicium is a Roblox script executor with a frameless dark UI, a Lua-aware code editor, one-click injection, and automatic backend updates ‚ÄĒ all in a single lightweight executable.

> **Closed source.** Binaries are available on the [releases page](https://github.com/miyamura00not/Silicium-web/releases/latest).

---

## Features

- **Instant injection** ‚ÄĒ attaches to Roblox in one click
- **AvalonEdit code editor** ‚ÄĒ full Lua syntax highlighting with Waylay orange theme
- **Auto-update** ‚ÄĒ backend files download automatically on launch
- **Encrypted IPC** ‚ÄĒ scripts sent over AES-encrypted named pipes
- **In-game notifications** ‚ÄĒ Roblox notification confirms inject
- **Minimal UI** ‚ÄĒ frameless, transparent window with custom chrome

## Usage

1. Launch `Silicium.exe`
2. Open Roblox and join a game
3. Click **Inject** ‚ÄĒ wait for the in-game notification
4. Paste your script into the editor
5. Click **Execute**
